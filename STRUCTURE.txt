═══════════════════════════════════════════════════════════

توضیحات فایل‌ها:
─────────────────

📄 index.html
├─ فایل اصلی سایت
├─ شامل سایدبار فصول
├─ مدیریت بارگذاری فصول
└─ صفحه خوش‌آمدگویی

📁 chapters/
├─ تمام فایل‌های HTML فصول در اینجا قرار می‌گیرند
├─ هر فایل باید نام‌گذاری دقیق داشته باشد (chapterXX.html)
└─ فایل‌ها باید HTML کامل باشند (با head و body)

📄 README.md
├─ راهنمای جامع پروژه
├─ نحوه نصب و راه‌اندازی
├─ عیب‌یابی مشکلات
└─ راهنمای GitHub Pages

📄 QUICK_START.md
├─ راهنمای سریع گام‌به‌گام
├─ برای شروع سریع کار
└─ سوالات متداول

📄 chapter01_template.html
├─ قالب نمونه برای ساخت فصول جدید
├─ شامل ساختار پیشنهادی
└─ می‌توانید کپی کرده و ویرایش کنید


═══════════════════════════════════════════════════════════

نکات مهم:
──────────

✅ نام‌گذاری دقیق فایل‌ها
   - chapter01.html نه Chapter01.html
   - chapters/ نه Chapters/

✅ ساختار پوشه‌ها
   - chapters در کنار index.html
   - نه داخل یا خارج از آن

✅ فایل‌های HTML
   - باید کامل باشند (DOCTYPE, html, head, body)
   - CDN ها و استایل‌ها در head قرار گیرند

✅ تنظیمات index.html
   - available: true برای فصول آماده
   - available: false برای فصول ناآماده


═══════════════════════════════════════════════════════════

مسیرهای فایل در کد:
────────────────────

در index.html:
└─ chapters/${chapter.file}
   مثال: chapters/chapter10.html

در GitHub Pages:
└─ https://USERNAME.github.io/REPO-NAME/
   └─ chapters/chapter10.html
